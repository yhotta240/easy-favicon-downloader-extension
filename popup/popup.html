概要

WEBサイトのファビコン（Favicon）を簡単にダウンロードできます

説明:

WEBサイトのファビコン（Favicon）を取得し，簡単にダウンロードできます．指定した URL のファビコンを複数のサイズで表示し，好みの形式（PNG，JPG，ICO）で保存可能です．

拡張機能の主な特徴:

  • ファビコンの取得: 指定したWEBサイトのファビコンを自動で取得し，表示します．
  • 複数サイズ対応: 16px，32px，48px，64px，96px，128px など，さまざまなサイズのファビコンを取得可能．
  • 形式の選択: PNG / JPG / ICO 形式でダウンロード可能．
  • アクティブタブの自動取得: 現在開いているWEBサイトのファビコンをワンクリックで取得できる．
  • 手動URL入力対応: 任意の URL を入力してファビコンを取得することも可能．

注意点:

  • ⚠ Google の Favicon API 依存: Google のファビコン取得API（https://www.google.com/s2/favicons）を使用しているため，一部のサイトでは正確なファビコンが取得できない可能性があります．
  • ⚠ Webサイトによる制約: 特定のサイトではファビコンのサイズやフォーマットが制限されている場合があります．
  • ⚠ ブラウザの権限設定: アクティブなタブの URL を取得するための権限を最小限に抑えていますが，一部の環境では動作しない可能性があります．

  • ファビコン：favicon
    ブラウザのタブや検索結果画面に表示される小さなアイコンのことを指す．「favorite icon（お気に入りのアイコン）」の略称．
  • PNG（Portable Network Graphics）:
    透過背景をサポートする画像形式
  • JPG（JPEG, Joint Photographic Experts Group）：
    圧縮率が高く，写真などに適した画像形式
  • ICO（Icon File）:
    Windows でアイコンとして使用されるファイル形式

1. Chrome ウェブストアから拡張機能をインストール

2. ファビコンをダウンロードする

① Chrome でファビコンを取得したい WEBサイト を開く
② その状態で拡張機能をクリックしてポップアップを開く
③ ファイル形式を変更する（オプション）
ポップアップ内の「ファイル形式の選択」で，ダウンロード形式を PNG，JPG，ICO から選択可能．
  • デフォルトは PNG 形式
  • 変更したい場合は ラジオボタンをクリック して選択
④ ファビコンをダウンロードする
  1. 各サイズ（16px，32px，48px，64px，96px，128px など）のプレビュー画像が表示される
  2. 取得したいサイズのファビコンの隣にある「ダウンロードボタン」をクリックしてダウンロード
補足: 特定のサイトのファビコンを取得したい場合，ポップアップ内のURL入力欄に WEBサイトのURL を入力することでファビコンを取得できます．

注意点

  • Google のファビコン取得API（https://www.google.com/s2/favicons）を使用しているため，一部のサイトでは正確なファビコンが取得できない可能性があります．
  • 特定のサイトではファビコンのサイズやフォーマットが制限されている場合があります．
  • アクティブなタブの URL を取得するための権限を最小限に抑えていますが，一部の環境では動作しない可能性があります．
